## 使用方法

首先，你需要安装一个Python(3.6以上版本)。

然后——

```
pip install git+https://github.com/RimoChan/kill3d.git
```

安装完成之后，你可以这样启动三次元人类屏蔽器——

```
python -m kill3d
```
